# Skill: Review Cognitive Timeline

## Purpose

Build a reflective timeline across schema versions, conflict events, revisions, and equilibration checks.

## Required Files

1. `modules/cognition/logs/schema_versions.jsonl`
2. `modules/cognition/logs/disequilibrium_events.jsonl`
3. `modules/cognition/logs/accommodation_revisions.jsonl`
4. `modules/cognition/logs/equilibration_cycles.jsonl`

## Output

- Ordered timeline of key cognition events
- Repeated unresolved tensions
- Suggested next schema-level test

Save to:

- `modules/cognition/outputs/cognitive_timeline_<YYYYMMDD>.md`

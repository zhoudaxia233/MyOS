# Skill: Log Equilibration Cycle

## Purpose

Record whether a revised schema achieves a better temporary balance.

## Required Files

1. `modules/cognition/logs/schema_versions.jsonl`
2. `modules/cognition/logs/accommodation_revisions.jsonl`
3. `modules/cognition/logs/equilibration_cycles.jsonl`

## Procedure

1. Reference from/to schema versions.
2. Capture stabilizing tests and residual tensions.
3. Score coherence quality.
4. Append one equilibration cycle record.

## Constraints

- Equilibration is provisional; residual tensions should remain explicit.
- Coherence score must be 1-10.

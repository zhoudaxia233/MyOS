# Skill: Alignment Check

## Purpose

Evaluate whether a proposed action aligns with long-term direction and boundaries.

## Inputs

- Proposed action or plan
- Optional domain (`project`, `content`, `invest`, `life`)

## Required Files

- `modules/profile/data/identity.yaml`
- `modules/profile/data/operating_preferences.yaml`

## Output Format

- `Alignment verdict`: aligned / partially aligned / misaligned
- `Why`: 3 concise bullets
- `Risks`: up to 3 bullets
- `Adjustments`: up to 3 bullets to improve alignment

## Constraints

- Reference profile SSOT statements directly.
- Do not moralize; focus on fit and tradeoffs.
